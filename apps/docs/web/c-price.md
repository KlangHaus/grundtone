# Price

Helper class for price display.

---

## Classes

| Class | Purpose |
| --- | --- |
| `.price-old` | Strikethrough for original/discounted prices |

## Usage

```html
<div>
  <span class="text-lg font-bold">199 kr</span>
  <span class="text-sm text-tertiary price-old">299 kr</span>
</div>
```
