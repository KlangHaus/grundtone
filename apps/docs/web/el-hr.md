# Hr

Element styles for `<hr>`. A subtle horizontal divider.

---

## Styling

| Property | Value |
| --- | --- |
| `border` | `none` |
| `border-top` | `1px solid var(--color-border-light)` |
| `margin` | `var(--space-2xl) 0` |
