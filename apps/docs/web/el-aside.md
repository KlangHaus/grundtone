# Aside

Semantic reference for `<aside>`.

---

## Usage

```html
<aside>
  <h3>Related</h3>
  <p>Tangentially related content like a sidebar or pull quote.</p>
</aside>
```

The `<aside>` element represents content indirectly related to the main content — sidebars, pull quotes, or related links.

## Accessibility

- Creates a `complementary` landmark role automatically.
- Use `aria-label` or `aria-labelledby` when multiple asides appear on the same page.
