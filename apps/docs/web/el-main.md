# Main

Semantic reference for `<main>`.

---

## Usage

```html
<main id="main">
  <h1>Page Title</h1>
  <p>The dominant content of the page.</p>
</main>
```

The `<main>` element wraps the primary content of the page. Use one per page.

## Accessibility

- Creates the `main` landmark role automatically.
- Pair with a skip link (`<a href="#main" class="skip-link">`) so keyboard users can bypass navigation.
