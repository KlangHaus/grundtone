# Article

Semantic reference for `<article>`.

---

## Usage

```html
<article>
  <h2>Article Title</h2>
  <p>Self-contained content such as a blog post or product card.</p>
</article>
```

The `<article>` element represents a self-contained composition intended to be independently distributable or reusable (e.g., in syndication) — a blog entry, a product card, a forum post, or a user-submitted comment.

## Accessibility

- Each `<article>` creates an `article` landmark role automatically.
- Use a heading inside the article for assistive-technology navigation.
- Nest `<article>` elements to represent comments or sub-items related to the parent article.
