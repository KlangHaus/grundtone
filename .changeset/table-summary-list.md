Add Table and Summary List components

**Table** (`GTTable`): Data table with sorting, selection, responsive stacking, and variants
(default, borderless, zebra, compact). Includes vanilla JS behavior for HTML-only usage.

**Summary List** (`GTSummaryList` + `GTSummaryItem`): Key-value pair display with `<dl>` semantics,
optional actions, and a card variant. Replaces the old `.structured-list` CSS.
